Yurucommu Namespace

This is the base namespace for Yurucommu's ActivityPub extensions.

JSON-LD Contexts

The combined context document covering all extensions:

https://yurucommu.com/ns/context.jsonld

Extension Namespaces

Extension Namespace Context Spec
Story https://yurucommu.com/ns/story# story/context.jsonld Spec
APC (Communities) https://yurucommu.com/ns/apc# apc/context.jsonld Spec

Usage

Include the combined context alongside the standard ActivityStreams context:

{
  "@context": [
    "https://www.w3.org/ns/activitystreams",
    "https://yurucommu.com/ns/context.jsonld"
  ],
  "type": "Note",
  "content": "Hello, Fediverse!"
}

Or include only the extension you need:

{
  "@context": [
    "https://www.w3.org/ns/activitystreams",
    "https://yurucommu.com/ns/story/context.jsonld"
  ],
  "type": ["Story", "Note"],
  "content": "...",
  "displayDuration": "PT15S"
}

Interoperability

Implementations that do not recognize Yurucommu's extension properties can safely ignore them per JSON-LD processing rules. All extensions are additive to standard ActivityPub / Activity Streams 2.0.
